# Calendar For Kakao Pay

100% self-developed all about The Calendar.

Didn't use any other UI Library.

# Start Application

## Tested In 
* NodeJS Version 10.15.3
* YARN Version 1.16.0

## Commands
```
$ yarn install
$ yarn start
```
